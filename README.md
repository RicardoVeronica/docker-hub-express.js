# This image will help you to start with JavaScript Express Microframework

### Just clone the repo
```
$ git clone https://github.com/RicardoVeronica/docker-hub-express.js.git
```

### Install packages
```
$ npm i
```

### Run the image
```
$ docker-compose up -d
```

### Open your browser in
0.0.0.0:3000

### Start to code

## Happy coding
