# This image will help you to start with JavaScript Express Microframework

### Just pull the image
```
$ docker pull ricardoveronica/express-app:1.0.0
```

### Run the image
```
$ docker-compose run --rm ricardoveronica/express-app:1.0.0
```

### Open your browser in
0.0.0.0:3000

### Start to code

## Happy coding
